# AFK × Etalon landing

Статический редизайн лендинга для предложения сотрудникам АФК «Система».

## Посмотреть онлайн

- [Открыть быстрый предпросмотр](https://htmlpreview.github.io/?https://github.com/OSKRENH/etaon/blob/main/afk-landing/index.html)
- GitHub Pages после активации репозитория: `https://oskrenh.github.io/etaon/`

## Что внутри

- адаптивная desktop-first сетка с максимальной шириной 1440 px;
- новый hero-блок с архитектурной SVG-композицией без тяжёлых изображений;
- компактное сравнение ипотечных программ и рассрочки;
- форма заявки с клиентской валидацией;
- мобильное меню, доступные состояния фокуса и поддержка `prefers-reduced-motion`;
- юридический дисклеймер и ссылки на официальный сайт.

## Локальный запуск

Откройте `index.html` напрямую либо запустите любой статический сервер:

```bash
python -m http.server 8080
```

Затем откройте `http://localhost:8080`.

## Подключение формы

Сейчас форма выполняет клиентскую проверку и показывает демонстрационный статус. Для production нужно заменить обработчик submit в `app.js` на отправку в CRM, Tilda Webhook или корпоративный API.

## Важное

Условия ипотечных программ перенесены с действующего лендинга и должны быть подтверждены ответственным менеджером перед публикацией.
